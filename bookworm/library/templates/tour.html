{% trans "Step 1: Adding a book" %}

{% blocktrans %} Select the Add a book link from the upper-right corner of every Bookworm page. {% endblocktrans %}

{% blocktrans %} Then click the Browse button. {% endblocktrans %}

{% blocktrans %} Your computer will prompt you to select a file to upload. This should be an ebook file that ends in .epub. {% endblocktrans %}

{% trans "Click " %} {% trans "Add to my library" %}.

{% blocktrans %} Your book is now available! {% endblocktrans %}

{% blocktrans %}From the main library screen you can view the list of books you've added, re-sort it by date, title or author, and delete unwanted books. {% endblocktrans %}

{% blocktrans %}Step 2: Reading a book {% endblocktrans %}

{% blocktrans %}Click on the book's title from your library. {% endblocktrans %}

{% trans "The book will automatically open to the first page." %}

{% blocktrans %} The left side of the screen shows the book's Table of Contents. The current chapter you're reading will be in large gray type. {% endblocktrans %}

{% blocktrans %} From the Table of Contents you can always click to Download your book in its original format, or click on Book information to see details about the book's copyright status, publisher, and other editions available online. {% endblocktrans %}

{% trans "The Book Information page" %}

{% blocktrans %} Publisher, language, copyright and other information supplied with the ebook can be read here. {% endblocktrans %}

{% blocktrans %} Bookworm will also automatically look up other editions of this work in Google Book Search. {% endblocktrans %}

{% blocktrans %}Moving through a book {% endblocktrans %}

{% blocktrans %} Each page of your book will have Next and/or Previous arrows to move between pages. You can also always jump to another part or chapter by clicking on the Table of Contents. {% endblocktrans %}

{% blocktrans %}Step 3: Searching your library {% endblocktrans %}

{% blocktrans %} If you've loaded books into your library, you'll see a search box at the top left of most Bookworm pages. Type a word or phrase in the box to find all occurrences of it in your library. {% endblocktrans %}

{% blocktrans %} Chapters in your books which contain this term will be displayed on the next page. Your search word or words will be highlighted in bold. Click on the book title to go to the matching chapter. {% endblocktrans %}

{% blocktrans %}Step 4: Take your book on the road {% endblocktrans %}

{% blocktrans %}Bookworm on your mobile phone {% endblocktrans %}

{% blocktrans %} If your phone has access to the web, then you can read your Bookworm books anywhere. Just point your phone browser to m.bookworm.oreilly.com. This version of Bookworm has been specially adapted for small devices. {% endblocktrans %}

{% blocktrans %}Stanza on the iPhone {% endblocktrans %}

{% blocktrans %} When viewed in Safari on the iPhone, Bookworm can export your books directly to Stanza. From your library list, click on the ⇧ arrow next to a title to add the book to your personal Stanza library. Step-by-step instructions on the Stanza site. {% endblocktrans %}

{% blocktrans %}Download to the Sony Reader or iLiad {% endblocktrans %}

{% blocktrans %} By clicking on the Download this book link in the Table of Contents, you can retrieve the original version of the ebook at any time. Follow the instructions in your ereader instructions for adding the book to your device. {% endblocktrans %}

{% blocktrans %}The Amazon Kindle cannot read ePub books at this time. {% endblocktrans %}
