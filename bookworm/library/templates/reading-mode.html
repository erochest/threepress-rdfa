{{ _("How to turn on Reading Mode") }}

{% blocktrans %}Reading Mode can be activated by clicking on the icon displaying a stack of pages. This icon will appear on any book page, in the upper right of the screen.{% endblocktrans %}

{{ _("Turning off Reading Mode") }}

{% blocktrans %}To return to the normal book display, click on the same icon as it appears in the Reading Mode layout. It will also appear in the upper right of the screen.{% endblocktrans %}

{{ _("Navigating Reading Mode") }}

{% blocktrans %}Reading Mode removes most of the Bookworm headers and footers and suppresses the display of the table of contents. Instead, the content is laid out in the center of the screen with ample whitespace to let you focus on reading.{% endblocktrans %}

{% blocktrans %}In place of cumbersome "next" and "previous" links, Reading Mode allows you to click anywhere in the right margin to turn to the next page, and click anywhere in the left margin to turn to the previous page.{% endblocktrans %}

{% blocktrans %}Bookworm provides subtle hints on how to operate the navigation. If you hover your mouse pointer over the margin areas, light-gray indicators will fade in to show that you are in a clickable navigation area.{% endblocktrans %}

{{ _("Go back to your library") }}

{% blocktrans %}To exit Reading Mode and return to the main library page, click on the arrow at the top or bottom next to your book's title.{% endblocktrans %}

{% blocktrans %}Bookworm will remember whether you prefer to use Reading Mode or not.{% endblocktrans %}

{{ _("Customizations") }}

{% blocktrans %}Both standard mode and Reading Mode provide methods to customize the fonts and font size of the ebooks. You can use use a sans-serif or serif font, and you can increase and decrease your font size. Bookworm will remember your font size preferences. These preferences will override whatever fonts and sizes were specified in the stylesheet that came with the ePub.{% endblocktrans %}

{% blocktrans %}In order to keep the user-interface simple, Bookworm does not provide more advanced font selectors such as picking a specific named font. However, it will use your preferred font by default while in Reading Mode. Use your browser's preferences menu to assign your favorite font as the default.{% endblocktrans %}
