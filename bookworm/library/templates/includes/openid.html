{% trans "What is OpenID?" %}

{% blocktrans %} OpenID eliminates the need for multiple usernames across different websites, simplifying your online experience. It's a standard used by many different websites, including Yahoo. {% endblocktrans %}

{% blocktrans %} You may already have an OpenID if you have an account with any of these websites. Click on the name to get more information about the OpenID services they offer: {% endblocktrans %}

  • Yahoo
  • {% blocktrans %}AOL or AIM users: Your OpenID is openid.aol.com/your-screenname {% endblocktrans %}
  • Wordpress
  • Basecamp, Highrise {% trans "or" %} Backpack
  • Livejournal: {% blocktrans %}Your OpenID is{% endblocktrans %} your-username.livejournal.com

{% blocktrans %} If you don't have an account with any of these services, or don't want to use them to manage your online identity, try MyOpenID. {% endblocktrans %}

{% blocktrans %} The official page for OpenID is http://openid.net/. {% endblocktrans %}
