About threepress

Updated April, 2008

threepress is a project to enable new revenue streams for print publishers using free and open tools for the web.

Projects

threepress is hosting two projects right now in varying stages of development:

  1. tei2epub is a standalone Python application for converting TEI XML into OPS 2.0.

    TEI (Text Encoding Initiative) is one of the leading standard encoding schemes for electronic text. OPS 2.0 is an open standard ebook format developed by the International Digital Publishing Forum. OPS files can be read with Adobe Digital Editions and, in the near future, portable digital readers such as the Kindle.

  2. threepress search is a Django application which includes utilities to index TEI content, serve carefully-formatted online texts, and provide ebook versions in various open formats.

Both projects are licensed under the GPL (except where different licenses apply to included components) and are available for checkout from Google Code at the threepress source repository.

The threepress sites includes fully-implemented samples of the above applications, with public domain content available for any use.

Where is the free book content from?

All books in the demonstration portion of the site are public domain and have been acquired in accordance with the guidelines set by the organizations.
