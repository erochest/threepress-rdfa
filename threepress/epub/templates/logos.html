ePub logos

Free, public domain, in PNG and Adobe Illustrator (AI) format

These logos representing the International Digital Publishing Forum (IDPF) ePub ebook format are in the public domain and can be used for any purpose with no attribution required. Alternate versions of the logos can be added here by contacting liza@threepress.org. SVG versions are especially encouraged.

These logos are in no way affiliated with the IDPF itself and have not been sanctioned by them.

Complete set of logos as an Adobe Illustrator file: epub-logos-ai.zip.

Right-click on each file to download, or download a complete set of logos in PNG format: epub-logos-png.zip.
